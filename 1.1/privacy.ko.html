🔒 Mindrafts 개인정보 처리방침

마지막 업데이트: 2025년 11월 15일

Mindrafts(이하 "앱", "당사")에 오신 것을 환영합니다.
당사는 이용자의 개인정보와 데이터 보안을 매우 중요하게 생각합니다. 본 방침은 로컬 AI 기능과 관련된 데이터를 포함해 개인정보를 어떻게 처리하는지 설명합니다.
앱을 사용하기 전에 반드시 읽어 주세요.

I. 기본 원칙

  1. 본 앱은 로컬 저장소와 로컬 대규모 언어 모델(LLM) 추론을 기반으로 하는 노트·지식 관리 도구입니다.
  2. 당사는 노트 내용을 저장하기 위한 전용 백엔드 서버를 구축하지 않으며, 노트 데이터를 원격 서버로 업로드하지 않습니다.
  3. 앱에서 생성된 모든 데이터는 오직 사용자 디바이스에만 저장되며, 사용자가 직접 관리합니다.
  4. 구독 구매·복원·검증(Apple 인앱 결제 시스템 이용)에 필요한 통신을 제외하고, 노트 내용이나 개인 식별 정보를 제3자 서버로 전송하지 않습니다.

II. 수집 정보

현재 버전의 Mindrafts는 개발자가 소유한 서버를 통해 아래 정보를 능동적으로 수집하거나 업로드하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 기기 모델, 운영체제 버전, 고유 식별자 등 디바이스 정보
  • GPS, IP 주소 등 위치 정보
  • 실행 횟수, 동작 로그, 크래시 로그 등 사용 패턴
  • 노트·태그·이미지·첨부 파일 등의 콘텐츠

이 모든 데이터는 로컬 디바이스에만 존재하며, 사용자가 직접 내보내거나 공유하지 않는 이상 외부로 나가지 않습니다.

⚠ 구독 및 결제 정보에 대하여
App Store / Apple 인앱 결제를 통해 구독을 구매·갱신·복원할 때:

  • 카드 정보나 Apple ID와 같은 결제 데이터는 Apple이 직접 처리하며, Apple의 개인정보 보호정책이 적용됩니다.
  • 당사는 시스템 API를 통해 구독 상품 종류, 유효 기간 등 권한 확인에 필요한 최소한의 정보만을 얻습니다.
  • 카드 세부정보를 수집하거나 당사 서버에 저장하지 않습니다.

III. 데이터 저장 및 보안

  1. 앱 데이터(노트, 태그, 설정, 로컬 LLM 추론 중간 결과 등)는 오직 사용자 디바이스에서만 저장·처리됩니다.
  2. 당사는 원격으로 사용자 데이터를 읽거나 개발자 서버에 업로드하지 않습니다.
  3. LLM 추론은 전적으로 디바이스에서 수행되며, 입력 텍스트와 관련 컨텍스트는 결과 생성을 위해서만 사용되고 원격 모델이나 클라우드 AI로 전송되지 않습니다.
  4. 데이터가 전적으로 사용자 통제 하에 있으므로 다음을 권장합니다. - 정기적인 백업으로 데이터 손실을 예방하세요. - 비밀번호, 생체 인증, 암호화 저장 등 OS 보안 기능을 활성화하세요. - 민감한 정보가 포함된 노트를 공유·내보낼 때는 각별히 주의하세요.
  5. 앱을 삭제하거나 데이터 초기화 시 관련 내용은 영구 삭제되며 복구할 수 없습니다.

IV. 제3자 서비스 및 의존성

  1. 현재 Mindrafts는 오픈소스 프로젝트 AFFiNE의 일부 코드를 사용하지만, AFFiNE 공식 온라인 서비스나 클라우드 동기화 기능을 연동하지 않습니다.
  2. 광고 추적 SDK나 분석 도구 같은 제3자 SDK를 포함하지 않으며, 광고/행동 분석 목적의 데이터 업로드도 없습니다.
  3. Apple 인앱 결제·구독 관련 사항: - App Store / StoreKit 메커니즘으로 구독 구매, 복원, 검증을 처리합니다. - 이 과정에서 Apple 서버와 필요한 통신이 발생합니다. - 결제·구독 데이터는 Apple 정책의 적용을 받으며 당사 통제 밖에 있습니다. - 유효 구독 여부 확인에 필요한 최소한의 정보만 로컬에 기록하며, 당사 서버로 전송하지 않습니다.
  4. 향후 클라우드 동기화, 온라인 AI, 크래시 분석, 통계 등 새로운 제3자 서비스가 도입될 경우, 본 방침을 업데이트하고 기능 활성화 전에 명시적 동의를 받겠습니다.

V. 로컬 AI(LLM) 기능

  1. 내장 LLM은 디바이스에서 실행되어 자동 완성, 문장 다듬기, 요약, 번역 등의 기능을 제공합니다.
  2. AI 기능을 호출하면 로컬 메모리에서 일시적으로 다음을 처리합니다. - 입력 중인 텍스트 - 응답 생성을 위해 필요한 최소한의 관련 컨텍스트(예: 현재 노트의 일부)
  3. 이 데이터는 로컬 추론 과정에서만 사용되며 제3자 서버나 클라우드 모델로 업로드되지 않습니다.
  4. 로컬 추론에 사용된 구체적 입력을 원격으로 기록하거나 업로드하지 않습니다.
  5. LLM의 확률적 특성상 결과는 다음과 같을 수 있습니다. - 완전하지 않거나 최신 정보가 아닐 수 있음 - 사실 오류나 누락을 포함할 수 있음 - 학습 데이터의 편향을 반영할 수 있음
    따라서 모든 응답이 정확하거나 모든 상황에 적합하다고 보장할 수 없으니 반드시 직접 검토하십시오.

VI. 이용자 권리

앱은 자체 서버를 통해 개인 식별 정보를 수집하지 않으므로, 이용자는 로컬 데이터에 대한 완전한 통제권을 가집니다. 다음이 가능합니다.

  • 언제든 노트를 생성·편집·삭제
  • 앱 또는 시스템 기능을 통한 데이터 내보내기/백업
  • 앱 삭제 또는 로컬 데이터 초기화로 모든 콘텐츠 제거
  • 별도 요청 없이도 «데이터 삭제» 및 «옵트아웃» 권리 행사

향후 원격 저장소나 계정 시스템이 필요할 경우, 사전 동의를 구하고 접근·정정·삭제 권리를 보장합니다.

VII. 아동 개인정보 보호

본 앱은 일반 사용자를 대상으로 하며, 아동을 위한 특별 서비스를 제공하지 않습니다.
18세 미만 미성년자의 식별 가능한 정보를 의도적으로 수집하거나 저장하지 않습니다.
현재 완전 로컬 구조에서는 미성년자가 생성한 데이터 역시 해당 디바이스에만 남으며, 보호자가 관리합니다.

VIII. 정책 업데이트

  1. 이 개인정보 처리방침은 앱 버전과 함께 자동으로 갱신되며, 최신 빌드에 포함됩니다.
  2. 별도 팝업이나 이메일로 업데이트를 통지하지 않으니, 앱 업데이트 후 최신 버전을 확인해 주세요.
  3. 향후 계정, 클라우드 동기화, 온라인 AI, 분석 도구 등 새로운 온라인/클라우드 기능 도입 시, 명확하게 고지하고 사용 전 새로운 정책에 동의하도록 요청합니다.

IX. 면책 조항 (AI 관련)

  1. 로컬 LLM 기능은 기록 및 글쓰기 효율을 높이기 위한 것으로, 전문적인 조언을 제공하지 않습니다.
  2. 특히 의료·법률·금융·투자·안전 등 고위험 영역에서는 AI가 생성한 내용을 반드시 자체 검증하고 필요 시 전문가와 상담하세요.
  3. AI 생성 콘텐츠를 맹신하거나 부적절하게 사용하여 발생하는 결과(경제적 손실, 인적 피해, 데이터 유출 등)에 대한 책임은 사용자에게 있습니다。
